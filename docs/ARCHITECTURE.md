## 组件交互图

```
媒体应用 (网易云、QQ音乐等)
    │
    ├─ 发送媒体通知
    │
    ↓
MediaNotificationListener
    │
    ├─ 解析通知
    ├─ 保存媒体信息
    ├─ 发送本地广播
    │
    ↓
NowPlayingProvider
    │
    ├─ 车机查询媒体信息
    ├─ 伪装成帆书
    ├─ 返回媒体数据
    │
    ↓
车机系统
    │
    ├─ 显示媒体信息
    ├─ 用户点击控制按钮
    │
    ↓
DesktopCardReceiver
    │
    ├─ 接收控制广播
    ├─ 转换为媒体按键
    │
    ↓
AudioManager
    │
    ├─ 发送按键事件
    │
    ↓
媒体应用
    │
    └─ 执行相应操作 (播放/暂停/下一曲)
```

## 数据流向

```
用户播放音乐
    ↓
系统发送媒体通知
    ↓
MediaNotificationListener.onNotificationPosted()
    ├─ 检查是否是媒体应用
    ├─ 解析通知内容
    ├─ 保存到 currentMedia (synchronized)
    ├─ 发送本地广播
    └─ 通知 NowPlayingProvider
    ↓
车机系统查询媒体信息
    ↓
NowPlayingProvider.query()
    ├─ 获取 currentMedia
    ├─ 构建 MatrixCursor
    ├─ 伪装成帆书应用
    └─ 返回数据
    ↓
车机显示媒体信息
    ↓
用户点击车机控制按钮
    ↓
车机发送广播
    ↓
DesktopCardReceiver.onReceive()
    ├─ 解析广播内容
    ├─ 转换为媒体按键
    └─ 调用 sendMediaKey()
    ↓
AudioManager.dispatchMediaKeyEvent()
    ↓
媒体应用接收按键事件
    ↓
媒体应用执行操作
    ↓
系统发送新的媒体通知 (循环)
```

## 生命周期

### 应用启动
```
1. DBToolApplication.onCreate()
   - 设置全局异常处理
   - 创建通知通道

2. MainActivity.onCreate()
   - 初始化 UI
   - 检查权限状态
   - 显示当前状态

3. BootReceiver 监听系统事件
   - 设备启动
   - 屏幕打开
   - 电源连接等

4. VehicleCoreService 启动
   - 创建前台通知
   - 开始监听媒体通知
```

### 运行时
```
1. MediaNotificationListener 持续监听
   - 媒体应用发送通知
   - 解析并保存信息

2. NowPlayingProvider 等待查询
   - 车机查询媒体信息
   - 返回伪装后的数据

3. DesktopCardReceiver 等待控制
   - 车机发送控制广播
   - 转换为媒体按键
   - 发送给媒体应用
```

### 应用关闭
```
1. MainActivity.onDestroy()
   - 清理 UI 资源

2. VehicleCoreService 继续运行
   - 前台服务不会被杀死
   - 继续监听媒体通知

3. 系统事件触发
   - 自动重启服务
```

## 权限模型

```
系统权限
├─ INTERNET
├─ ACCESS_NETWORK_STATE
├─ WAKE_LOCK
├─ FOREGROUND_SERVICE
├─ READ_EXTERNAL_STORAGE
├─ WRITE_EXTERNAL_STORAGE
├─ BIND_NOTIFICATION_LISTENER_SERVICE
├─ BIND_ACCESSIBILITY_SERVICE
└─ FOREGROUND_SERVICE_DATA_SYNC

自定义权限 (signature 级别)
├─ ACCESS_MEDIA_INFO
├─ MEDIA_CONTROL
└─ DYNAMIC_RECEIVER_NOT_EXPORTED_PERMISSION
```

## 关键设计模式

| 模式 | 应用 | 说明 |
|------|------|------|
| **单例** | MediaNotificationListener.currentMedia | 全局媒体信息存储 |
| **观察者** | NotificationListenerService | 监听系统通知 |
| **适配器** | NowPlayingProvider | 适配不同车机协议 |
| **工厂** | VehicleCoreService | 创建前台服务 |
| **代理** | DesktopCardReceiver | 代理车机控制命令 |
| **弱引用** | DBToolAccessibilityService | 防止内存泄漏 |
| **同步锁** | MediaNotificationListener | 线程安全 |
